"impress.js"とは

CSS3を活用したオープンソース(MITライセンス)のプレゼンテーションJavaScriptライブラリ。
prezi.comのようなスライドを作ることが可能。

  • preziのようにx,y軸の移動だけでなく、z軸の移動も可能

"impress.js" $\times$ "MathJax" 目標

  1. 動作が軽快
  2. 日本語が使える
  3. オフラインでも使える

作成準備

"impress.js"のDL

impress.jsのGitHubリポジトリから"Download ZIP"で全ファイルをDL("git clone"ではimpress.jsしかDLできなかったので...)
"impress.js"の"index.html"をベースに作っていきます。

"MathJax"のDL

MathJaxのGitHubリポジトリからMathJaxをDLして、impress.jsのフォルダの適当な場所に置く。

MathJaxに命を吹き込む

impress.jsのindex.htmlの < head > ... < /head > に以下を加えます。 

< script type="text/javascript"src="MathJaxへのパス/MathJax.js?config=TeX-AMS-MML_HTMLorMML"> < /script >
< script type="text/x-mathjax-config" > MathJax.Hub.Config({ tex2jax: { inlineMath: [['$','$']] } });< /script >

これでMathJaxが使えるようになった。

MathJaxを軽量化する

MathJaxには30,000を超える数式のpng画像が含まれています。しかし、この画像は古いブラウザに対応するためのものでよほど古いブラウザで使用しない限り、消してしまっても構いません。なので、今回は消してしまいます。 MathJaxのfontsフォルダ以下のpng画像をすべて削除します。 これで大分動作が軽くなります。

フォントの設定変更

Webフォントをローカルフォントに変更

  1. Google Fontsで"Open Sans", "PT Sans", "PT Serif" を検索,ダウンロードして、任意の場所に保存する
  2. cssに以下の内容を"Open Sans", "PT Sans", "PT Serif"のそれぞれに対して記述する(以下は"Open Sans"の例)
@font-face {
    font-family: Open Sans;
    src: url('Open_Sansへのパス/OpenSans-Regular.ttf') format("truetype");
}

日本語フォントを追加

IPAexフォントをここからDLし任意の場所に保存し、cssに先ほどと同じようにfont-fmilyを追加する。 

@font-face {
    font-family: IPAex Mincho;
    src: url('IPAexへのパス/ipaexm.ttf') format("truetype");
}

index.htmlの中の"Open Sans", "PT Sans", "PT Serif"と書かれているところを探し出し、"Sans"と書いてあるところには"IPAexゴシック"を、"Serif"と書いてあるところには"IPAex明朝"を追加する。

例

font-family: 'PT Serif', 'IPAex Mincho', georgia, serif;
xfont-family: 'PT Sans', 'IPAex Gothic', sans-serif;

ただし、'IPAex Mincho','IPAex Gothic'にはcssに記述した名前で書く。
